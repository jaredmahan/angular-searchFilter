Introduction
The Angular Search Filter module is a easy robust way to manage filtering in your angular application. This module utilizes the angular workflow allowing you to add filters in your html. While filtering the module spits out a filter object includes a builtin ODATA query generated from your filters.

Things this module provides to you:
  • Works well with Bootstrap
  • Designed for use with angular
  • Builtin OData filtering
  • Intuitive filters based on type
  • Type Templating
  • Filter API that allows the developer to manage filters as they please
